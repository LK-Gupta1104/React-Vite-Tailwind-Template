# React + Vite + Tailwind Starter Template

A clean starter template for building React applications using Vite and Tailwind CSS v4.

## Features

* React 19
* Vite 8
* Tailwind CSS v4
* ESLint Configuration
* Fast Development Server
* Clean Project Structure

## Getting Started

### Clone the Repository

```bash
git clone <your-repository-url>
cd react-vite-tailwind-template
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```text
src/
├── App.jsx
├── index.css
└── main.jsx

public/

vite.config.js
package.json
```

## Tailwind Setup

The project uses Tailwind CSS v4.

`src/index.css`

```css
@import "tailwindcss";
```

## How to Use This Template

1. Clone or copy this template.
2. Rename the project folder.
3. Run:

```bash
npm install
npm run dev
```

4. Start building your application.
